This repository serves as a centralized showcase for my practical engineering projects, hardware simulations, and embedded hardware/software automation workflows.

## 🔌 Project 1: Microcontroller Peripheral Interfacing (LED & Fan Control)
A foundational firmware script written in C++ (Arduino framework) demonstrating input/output control loops over external peripherals using digital pins.

### 📌 Hardware Connection Reference
* **Digital Pin 12:** Connected to a 5V DC Cooling Fan via a relay/driver terminal.
* **Digital Pin 13:** Connected to an Indicator LED with a current-limiting resistor ($220\,\Omega$).
* **GND:** Tied to the common ground terminal of the circuit schematic layout.
